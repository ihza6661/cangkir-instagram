## CAROUSEL — Post #8 | Brand F&B → Website

---

### SLIDE 1 — Hook

**Teks utama:**
```
Followers ribuan.
Tapi pas dicari di Google —
tidak ada.
```

**Teks kecil / subtext:**
```
Ini masalah banyak brand F&B di Pontianak.
```

**Visual brief:**
```
Background gelap atau solid brand Cangkir.
Teks besar, bold, center.
Tidak perlu banyak elemen — biarkan teksnya yang bicara.
```

---

### SLIDE 2 — Problem

**Teks utama:**
```
Instagram itu untuk engagement.
Bukan untuk kredibilitas bisnis.
```

**Teks isi:**
```
Calon reseller, supplier, atau investor
tidak akan DM kamu duluan.

Mereka akan Google nama brand kamu.
Dan kalau tidak ada — mereka lanjut ke brand lain.
```

**Visual brief:**
```
Layout bersih, teks rata kiri.
Ilustrasi sederhana: search bar Google 
dengan hasil "No results" atau kosong.
```

---

### SLIDE 3 — Solution

**Teks utama:**
```
Website company profile
bukan sekadar tampilan.
Ini adalah alamat resmi brand kamu.
```

**Teks isi:**
```
✓ Muncul di Google saat nama brand dicari
✓ Menjawab pertanyaan B2B secara profesional
✓ Bisa dicantumkan di proposal, packaging, email bisnis
✓ Aktif 24 jam — tanpa harus balas DM
```

**Visual brief:**
```
Background terang / putih.
Checklist dengan ikon sederhana.
Tampilkan mockup website F&B di sisi kanan atau bawah.
```

---

### SLIDE 4 — CTA

**Teks utama:**
```
Siap terlihat profesional?
```

**Teks isi:**
```
Kami bantu brand F&B di Pontianak
punya website yang bekerja untuk bisnis kamu.

Konsultasi gratis — tidak ada kewajiban.
```

**Teks CTA:**
```
DM sekarang atau klik link di bio.
```

**Visual brief:**
```
Warna brand Cangkir, clean.
Logo Cangkir prominent.
Tampilkan nomor WhatsApp atau username Instagram.
Tombol/badge CTA yang jelas.
```
